# Exercício 2 da Fennec

Após clonar o repositorio baixe as dependências:
```shell
  npm i
```

Para inicializar o banco de dados use docker, rode esse comando:
```shell
  docker-compose up -d
```

Para iniciar com a aplicação, rode os seguintes comandos:
```shell
  npm run build
```

```shell
  npm run start
```
